# vagrant-nginx-2020
Proyecto que utiliza Vagrant para crear un entorno Nginx+PHP+MySQL

## Uso

Clonar el proyecto
```
git clone https://github.com/tics-frcu/vagrant-nginx-2020.git
```

Ingresar al directorio
```
cd vagrant-nginx-2020
```

Levantar el entorno
```
vagrant up
```
